## Архитектура

```
FAM-Core App
├── Frontend (React + TypeScript)
│   ├── Поле ввода
│   ├── Кнопка "Сохранить" → write_test_record
│   ├── Кнопка "Статус" → get_status
│   └── Кнопка "Прочитать" → get_version
│
└── Backend (Rust + Tauri)
    ├── Database (rusqlite + SQLCipher bundled)
    │   ├── init_db() - инициализация при старте
    │   ├── PRAGMA key - шифрование
    │   └── meta table - хранение данных
    │
    ├── Crypto (Argon2id)
    │   ├── generate_master_key() - 32 байта
    │   ├── derive_key() - деривация из пароля
    │   └── verify_key() - проверка ключа
    │
    └── API (HTTP клиент)
        └── make_request() - HTTP запросы
```
